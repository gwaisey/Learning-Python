# Learning Python 🐍

Welcome to my Python learning journey! As a Computer Science student and Software Developer typically focused on the PHP/Laravel ecosystem, this repository serves as a live documentation of my transition into Python.

## 🚀 Purpose
The goal of this repo is to build a solid foundation in Python by migrating logic from my previous projects (like Inventory Management) and exploring Python's unique capabilities in data processing and automation.

## 🛠️ Integrated Keyboard Inventory System
This system bridges a **Terminal CLI** and a **Web Dashboard** using a shared `inventory.json` database.

## 📁 Project Structure
- `scripts/`: Contains Python scripts for terminal-based data analytics.
- `web-app/`: Contains Flask-based web applications for visual dashboards.

### ⌨️ Keyboard Inventory Analytics
A script designed to manage and analyze stock for mechanical keyboard components (switches, keycaps, and kits).
- **Concepts used:** Lists, Dictionaries, Functions, and F-string formatting.
- **Features:** Automated stock status alerts and asset valuation.

## 🛠️ Tech Stack
- **Environment:** GitHub Codespaces (Cloud Development)
- **Language:** Python 3.x
- **Tools:** VS Code

### How to Run:
```bash
# 1. Install dependencies
